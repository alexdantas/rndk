tawny-cdk
========

A Ruby version of the curses development kit.

Why tawny?  Because this is better than a ruby port.

Currently implemented widgets:
 * Alphalist
 * Button
 * Buttonbox
 * Calendar
 * Dialog
 * Entry
 * File Selector
 * Graph
 * Histogram
 * Item List
 * Label
 * Marquee
 * Menu
 * Multiple Line Entry
 * Radio List
 * Scale
 * Scrolling List
 * Scrolling Window 
 * Selection List
 * Slider
 * Template
 * Viewer

Original C project page: http://invisible-island.net/cdk/
